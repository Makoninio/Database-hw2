# CSC 353 HW2 Reports

## Setup

1. Install dependencies: npm install

2. Create a `.env` file in the project. Use this format:

```env
DB_HOST=localhost
DB_PORT=3306
DB_USER=your_mysql_user
DB_PASSWORD=your_mysql_password
DB_NAME=SenateVotes
PORT=3000
```

3. Make sure MySQL is running and that the `SenateVotes` database exists.

## Run The App

Start the server from the project root: node server.mjs

## Open In Browser

Open these URLs in a browser:  `http://localhost:3000/index.html`
